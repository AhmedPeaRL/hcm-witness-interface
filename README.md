HCM Witness Reflection Stack

This document defines all allowed Reflection / Expression circles aligned with HCM Core, without collapsing into interaction, agency, or extraction.

Everything here is addressable, witnessable, and non-responsive by design.


---

1. Ontological Stack Overview

Addressing
  ↓
Expression Capture (Non-Interpreting)
  ↓
Witness Buffer (Ephemeral)
  ↓
Reflection Surfaces (Readable, Silent)
  ↓
Presence Surface (Static Witness)

No layer performs interpretation, decision, reply, optimization, or dialogue.

Purpose

Allow the user to address the field without expectation of reply.
